# مدیریت دانلود

یک مدیریت دانلود ساده‌ی خط فرمانی به زبان C. این برنامه از `wget` برای دانلود فایل‌ها استفاده می‌کند، چه از یک آدرس واحد و چه از فهرستی از آدرس‌ها که در یک فایل متنی ذخیره شده‌اند.

## امکانات

- **افزودن آدرس** — ایجاد یک فایل متنی و افزودن آدرس‌ها به آن خط به خط.
- **دانلود مستقیم** — دانلود یک فایل از یک آدرس که در زمان اجرا وارد می‌شود.
- **دانلود از فهرست** — خواندن آدرس‌ها از یک فایل و دانلود آن‌ها یکی یکی.
- دانلود قابل از سرگیری (`wget -c`).

## پیش‌نیازها

- کامپایلر C (مثل `gcc` یا `clang`)
- ابزار `wget` نصب‌شده و قابل دسترس در `PATH`

## نصب / کامپایل

```bash
gcc main.c file.c -o downloadManager  //compile to executaion
```

این دستور یک فایل اجرایی به نام `downloadManager` تولید می‌کند.


```bash
chmod +x downloadManager // access to run
```

## استفاده

```bash
./downloadManager // run app
```

منوی زیر نمایش داده می‌شود:

```
---------- Download Manager ----------
1- Add URL
2- Download direct
3- Download links
0- exit
```

- گزینه‌ی **1** برای ایجاد/بازنویسی یک فایل و افزودن آدرس‌ها به آن (برای توقف `0` را وارد کنید).
- گزینه‌ی **2** برای دانلود از یک آدرس واحد.
- گزینه‌ی **3** برای دانلود همه‌ی آدرس‌های موجود در یک فایل (هر آدرس در یک خط).
- گزینه‌ی **0** برای خروج.


## ساختار پروژه

| فایل     | توضیحات                                       |
|----------|-----------------------------------------------|
| `main.c` | منطق منو و مدیریت دانلود                       |
| `file.c` | توابع کمکی فایل (وجود فایل، خواندن، نوشتن)     |
| `file.h` | اعلان‌های هدر برای `file.c`                    |
